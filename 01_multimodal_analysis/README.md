## SUMMARY

+ Describir las funcionalidades del Servicio de comprensión de contenido de IA de Azure.
+ Use el Servicio de comprensión de contenido de IA de Azure para crear un analizador de contenido.
+ Consuma un analizador de comprensión de contenido mediante la API de REST


+ Azure AI Content Understanding

## STEPS
+ Create an Azure AI services resource.
+ Define a Content Understanding schema for the information to be extracted. 
+ This can be based on a content sample and an analyzer template.
+ Build an analyzer based on the completed schema.
+ Use the analyzer to extract or generate fields from new content.
